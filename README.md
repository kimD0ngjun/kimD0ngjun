## 💬 Introduction

두 번의 부트캠프를 통해 프론트엔드를 이해하는 신입 백엔드 개발자입니다.

대용량 트래픽 제어 프로젝트에서 테스트 및 아키텍처 개선을 통한 성능 향상을 이끌어냈습니다.

새로운 스킬을 이론 학습에만 그치지 않고 직접 코드로 활용하고 참여한 프로젝트에 적용합니다.

현재의 성과에 안주하지 않고 계속 성장하여 팀의 성공에 기여하는 개발자가 되도록 노력합니다.
